# Camil Teraoui - Personal Portfolio

Minimal, premium single-page portfolio built with React, Tailwind CSS, and Framer Motion.

## Stack

- React (functional components)
- Tailwind CSS
- Framer Motion
- Vite

## Project Structure

```
src/
  components/
  sections/
  data/
  App.jsx
  main.jsx
  index.css
```

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Notes

- Content is intentionally placeholder-based for easy customization.
- Design direction is grayscale, minimal, and typography-led.
- Animations are subtle and implemented with Framer Motion.
